Security in My Day-to-Day

Workplace Habits

  • Lock my workstation when stepping away
  • Use strong, unique passwords
  • Enable 2FA on internal tools
  • Report suspicious emails or links immediately

Personal Security

  • Use a password manager
  • Be cautious with unknown websites and downloads
  • Regularly update software
